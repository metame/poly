# poly

The tool to end all tools

## Install

`$ cargo install --git https://github.com/metame/poly.git`

or clone repo and run:

`cargo install --path .`

## Usage

`$ poly <language> <command>`

e.g.

``` shell
$ poly rust build
You want to Build with Rust? Let me try to help you with that
cargo build
```
